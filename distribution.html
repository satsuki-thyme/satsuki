EmEditor

Markdown 用シンタックスハイライト

任意の Markdown ファイルを開き、メニューから ツール > 現在の設定のプロパティ > 強調(1) を開いて markdown.ESY をインポートしてください。

ダウンロード : markdown.ESY

Markdown 向けテーマ

任意の Markdown ファイルを開き、メニューから ツール > 現在の設定のプロパティ > 表示 を開いて「テーマ(T): 」の右にある「 > 」ボタンから markdown.eetheme をインポートしてください。

ダウンロード : markdown.eetheme

Markdown 用アウトライン設定表

markdonw-outline.md に従って手入力してください。入力するには EmEditor で任意の Markdown ファイルを開き、メニューから ツール > 現在の設定のプロパティ > アウトライン を開いてください。

ダウンロード : markdown-outline.md

サクラエディタ

Markdown 用シンタックスハイライト

任意の Markdown ファイルを開き、メニューから 設定 > タイプ別設定 を開いて 正規表現キーワード タブから markdown.rkw をインポートしてください。

ダウンロード : markdown.rkw

Markdown 向けテーマ

任意の Markdown ファイルを開き、メニューから 設定 > タイプ別設定 を開いて カラー タブから markdown.col をインポートしてください。

ダウンロード : markdown.col

Markdown 用設定

上記 2 設定を合わせてインポートすることができます。サクラエディタのメニューから 設定 > タイプ別設定一覧 を開いて markdown.ini をインポートしてください。

ダウンロード : markdown.ini

Mery

Markdown 用シンタックスハイライト

任意の Markdown ファイルを開き、メニューから 表示 > 編集モード > 編集モードの設定 を開いて Markdown を選択して（なければ新規作成） プロパティ ボタンをクリックし、 強調 タブから markdown.msy をインポートしてください。

ダウンロード : markdown.msy

Markdown 向けテーマ

メニューから ツール > オプション を開いて 表示 タブを開いて「テーマ(T): 」の右にある「 > 」ボタンから markdown.metheme をインポートしてください。

ダウンロード : markdown.metheme

Google 日本語入力

  • 「角（かど）」「角（つの）」など読み方の微妙な漢字

ダウンロード : google-japanese-input-ruby.txt
